How It Works

The 6-Agent Pipeline

👤

Context Agent

Interviews you to uncover hidden constraints — budget, timeline, preferences — ensuring no detail is missed.

🔍

Research Agent

Compiles real specs, prices, pros and cons for every option you're considering — grounded in facts.

⚖️

Analysis Agent

The core reasoning engine — builds a comparative matrix of trade-offs, strengths, and opportunity costs.

⚠️

Risk Agent

Predicts future pitfalls most AIs miss: "Can't train large models", "Market saturated in 2 years".

🧮

Decision Agent

Performs weighted multi-criteria scoring — Cost, Performance, Risk, Future — to produce a final score.

💬

Explanation Agent

Translates everything into a human-friendly explanation: WHY this option, and WHY NOT the others.
